# deliver_xss
This repository can be used to show the possibility of injecting Javascript by using jsdelivr.net

# How it works:
All the code from Github can also be served by jsdelivr.net.
So this repository can be found at:
https://cdn.jsdelivr.net/gh/MeneerHeijpaal/deliver_xss@main/<JS FILE>

To do a simple XSS check, do:
[https://cdn.jsdelivr.net/gh/MeneerHeijpaal/deliver_xss@main/simple.js](https://cdn.jsdelivr.net/gh/MeneerHeijpaal/deliver_xss@main/simple.js)
